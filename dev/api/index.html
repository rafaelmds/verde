API Reference¶

verde¶

The Verde public API.

Classes

verde.ScipyGridder([method, extra_args]) A scipy.interpolate based gridder for scalar Cartesian data.

Functions

verde.block_reduce(easting, northing, data, …) Apply a reduction operation to the data in blocks (windows).
verde.block_region(region, spacing[, adjust]) Divide a region into blocks and yield the boundaries of each block.
verde.get_region(easting, northing) Get the bounding region of the given coordinates.
verde.grid_coordinates(region[, shape, …]) Generate the coordinates for each point on a regular grid.
verde.inside(easting, northing, region[, …]) Determine which points fall inside a given region.
verde.profile_coordinates(point1, point2, size) Coordinates for a profile along a line between two points.
verde.scatter_points(region, size[, …]) Generate the coordinates for a random scatter of points.
verde.test([doctest, verbose, coverage]) Run the test suite.
